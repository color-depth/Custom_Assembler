Basic Assembler

add reg1 reg2 reg3

Performs reg1 = reg2 + reg3.
If the computation overflows, then the overflow flag is set

sub reg1 reg2 reg3

Performs reg1 = reg2 - reg3.
In case reg3 > reg2, 0 is written to reg1 and overflow flag is set.

mov reg1 $Imm

Performs reg1 = $Imm where Imm is a 8 bit value.

mov reg1 reg2

Performs reg1 = reg2.

ld reg1 mem_addr

Loads data from mem_addr into reg1.

st reg1 mem_addr

Stores data from reg1 to mem_addr.

mul reg1 reg2 reg3

Performs reg1 = reg2 x reg3.
If the computation overflows, then the overflow flag is set.

div reg3 reg4

Performs reg3/reg4.
Stores the quotient in R0 and the remainder in R1.

rs reg1 $Imm

Right shifts reg1 by $Imm, where $Imm is an 8 bit value.

ls reg1 $Imm

Left shifts reg1 by $Imm, where $Imm is an 8 bit value.

xor reg1 reg2 reg3

Performs bitwise XOR of reg2 and reg3. Stores the result in reg1.

or reg1 reg2 reg3

Performs bitwise OR of reg2 and reg3.
Stores the result in reg1.

and reg1 reg2 reg3

Performs bitwise AND of reg2 and reg3.
Stores the result in reg1.

not reg1 reg2

Performs bitwise NOT of reg2.
Stores the result in reg1.

cmp reg1 reg2

Compares reg1 and C reg2 and sets up the FLAGS register.

jmp mem_addr

Unconditional Jumps to mem_addr.

jlt mem_addr

Jump to mem_addr if the less than flag is set (less than flag = 1), where mem_addr is a memory address.

jgt mem_addr

Jump to mem_addr if the greater than flag is set (greater than flag = 1), where mem_addr is a memory address.

je mem_addr

Jump to mem_addr if the equal flag is set (equal flag = 1), where mem_addr is a memory address.

hlt

Halt Stops the machine from executing until reset
